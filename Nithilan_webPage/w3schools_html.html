Defines an acronym

19-Mar-2017 <address> Defines contact information for the author/owner of a document/article
19-Mar-2017 <b> Defines bold text

20-Mar-2017 <bdo> Overrides the current text direction
20-Mar-2017 <big> Not supported in HTML5. Use CSS instead.
Defines big text
20-Mar-2017 <blockquote> Defines a section that is quoted from another source
21-Mar-2017 <center> Not supported in HTML5. Use CSS instead.
Defines centered text
21-Mar-2017 <cite> Defines the title of a work
21-Mar-2017 <code> Defines a piece of computer code
21-Mar-2017 <del> Defines text that has been deleted from a document
22-Mar-2017 <dfn> Represents the defining instance of a term
22-Mar-2017 <em> Defines emphasized text 
22-Mar-2017 <font> Not supported in HTML5. Use CSS instead.
Defines font, color, and size for text
22-Mar-2017 <i> Defines a part of text in an alternate voice or mood
23-Mar-2017 <ins> Defines a text that has been inserted into a document
23-Mar-2017 <kbd> Defines keyboard input
23-Mar-2017 <mark> Defines marked/highlighted text
23-Mar-2017 <meter> Defines a scalar measurement within a known range (a gauge)
24-Mar-2017 <pre> Defines preformatted text

24-Mar-2017 <q> Defines a short quotation

25-Mar-2017 <s> Defines text that is no longer correct
25-Mar-2017 <samp> Defines sample output from a computer program
26-Mar-2017 <small> Defines smaller text
26-Mar-2017 <strike> Not supported in HTML5. Use <del> or <s> instead.
Defines strikethrough text
26-Mar-2017 <strong> Defines important text
26-Mar-2017 <sub> Defines subscripted text
27-Mar-2017 <sup> Defines superscripted text
27-Mar-2017 <time> Defines a date/time
27-Mar-2017 <tt> Not supported in HTML5. Use CSS instead.
Defines teletype text
27-Mar-2017 <u> Defines text that should be stylistically different from normal text
28-Mar-2017 <var> Defines a variable
28-Mar-2017 <wbr> Defines a possible line-break

Date Tag Description

01-Apr-2017 <frameset> Not supported in HTML5.
Defines a set of frames
01-Apr-2017 <noframes> Not supported in HTML5.
Defines an alternate content for users that do not support frames
